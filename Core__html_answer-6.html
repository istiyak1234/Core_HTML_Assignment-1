List of Tags

  1. h1 :- The HTML h1 tag defines the highest level or most important heading in the HTML document. This tag is also commonly referred to as the h1 element.

  2. div:- The div tag defines a division or a section in an HTML document. The div tag is used as a container for HTML elements - which is then styled with CSS or manipulated with JavaScript.

  3. p:- The p tag in HTML defines a paragraph. These have both opening and closing tags.

  4. table:- The table tag defines an HTML table.

  5. ul:- The ul tag defines an unordered (bulleted) list.
